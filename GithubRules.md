# Github instructions
## How to create a new branch
Run this command
````git checkout -b <new branch name>````

## How to switch to a specific branch
Run this command 
````git switch <branch name>````

## How to see all the remote branches
```git branch -r```

## How to install maven
Use this [link](https://maven.apache.org/install.html) https://maven.apache.org/install.html


## Rules for github
- No push to main
- For each sprint create a new branch
- Use pull request from the sprint branch to the main branch (potentially use approve functionality in github)
- In commit message specify the changes you have done
